# NexaApp — Full-Stack Auth & Dashboard

A professional, high-fidelity user management system built for high engagement and visual excellence. Designed with modern web aesthetics including Glassmorphism, Dynamic Backgrounds, and a seamless Dark/Light theme system.

## 🚀 Features

- **Multi-Step Signup**: Intuitive 3-step registration flow with real-time validation.
- **Secure Authentication**: Backend login system with password hashing (Bcrypt).
- **Interactive Dashboard**: Personalized user welcome screen with interest chips and account status.
- **Premium UI/UX**:
    - **Dual Theme Support**: Switch between Dark and Light modes on the fly.
    - **Glassmorphism Design**: Frosted glass effects and vibrant background blobs.
    - **Smooth Transitions**: Carefully crafted view switching with slide and fade animations.
    - **Password Strength Meter**: Real-time visual feedback on security quality.
- **Full-Stack Integration**: Node.js/Express backend connected to MongoDB Atlas.

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas (Mongoose)
- **Deployment**: netlify ready

## 📦 Setup & Installation

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Set up environment variables**:
   Create a `.env` file in the root with:
   ```env
   MONGODB_URI=your_mongodb_connection_string
   PORT=5000
   ```
4. **Run locally**:
   ```bash
   npm run dev
   ```
